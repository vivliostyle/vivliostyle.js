Basic CMYK Colors

Cyan text: device-cmyk(1 0 0 0)

Magenta text: device-cmyk(0 1 0 0)

Yellow text: device-cmyk(0 0 1 0)

Black text: device-cmyk(0 0 0 1)

With Alpha

Semi-transparent: device-cmyk(0.5 0.5 0.5 0.5 / 0.5)

Percentage Notation

Cyan with percent: device-cmyk(100% 0% 0% 0%)

Legacy Syntax (comma-separated)

Legacy cyan: device-cmyk(1, 0, 0, 0)

Legacy mixed: device-cmyk(0.3, 0.5, 0.7, 0.2)

Mixed Values

Mixed: device-cmyk(0.2 0.4 0.6 0.1)
