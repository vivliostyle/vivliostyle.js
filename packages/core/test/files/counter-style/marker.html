::marker pseudo-element

with counter()

li::marker { content: counter(list-item, cjk-decimal) "、"; }
  1. One
  2. Two
  3. Three
ol { list-style-type: cjk-decimal; }
  1. One
  2. Two
  3. Three

These two lists above should look the same.

with string

li::marker { content: "✅ "; }
  1. One
  2. Two
  3. Three
ol { list-style-type: "✅ "; }
  1. One
  2. Two
  3. Three

These two lists above should look the same.

with image

li::marker { content: url(…); }
  1. One
  2. Two
  3. Three
li::marker { content: url(…) " "; }
  1. One
  2. Two
  3. Three
ol { list-style-image: url(…); }
  1. One
  2. Two
  3. Three

These three lists above should look the same except the first one has no space between the image and the text.
