::marker pseudo-element

with counter()

li::marker { content: counter(list-item, cjk-decimal) "、"; }
  1. One
  2. Two
  3. Three
ol { list-style-type: cjk-decimal; }
  1. One
  2. Two
  3. Three

These two lists above should look the same.

with string

li::marker { content: "✅ "; }
  1. One
  2. Two
  3. Three
ol { list-style-type: "✅ "; }
  1. One
  2. Two
  3. Three

These two lists above should look the same.

with image

li::marker { content: url(…); }
  1. One
  2. Two
  3. Three
li::marker { content: url(…) " "; }
  1. One
  2. Two
  3. Three
ol { list-style-image: url(…); }
  1. One
  2. Two
  3. Three

These three lists above should look the same except the first one has no space between the image and the text.

with large font-size

li::marker { font-size: 2em; }
  1. One — markerのfont-sizeが2emでも行間が適切に広がること
  2. Two
  3. Three

Each list item above should have enough line height to accommodate the enlarged marker. The marker and the list item text should not overlap.

with large image

li::marker { content: url(… width="48") " "; }
  1. One — markerの画像が大きくても前の行と重ならないこと
  2. Two
  3. Three
ol { list-style-image: url(… width="48"); }
  1. One — markerの画像が大きくても前の行と重ならないこと
  2. Two
  3. Three

In the lists above, the large (48px) star image marker should not overlap with the preceding line. Each item's line box should be tall enough to contain the image.
